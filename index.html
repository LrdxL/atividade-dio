O que são Clients e Servers?

Definição de Clients:

Lado em que recebe a informação da internet.
Exemplos: PC, Smartphones e Tablets.

Conectado em um server é possivel então receber arquivos estáticos - HTML, CSS, JS e/ou imagens/áudios. Nessa ponta da conversa temos também os famosos Navegadores, que armazenam esses arquivos solicitados em cache para acelerar uma posterior requisição.

Navegadores

Definição: Programas criados por empresas para abrir/executar arquivos específicos.
Exemplos: Chrome, Firefox, Safari e Thor.

Principais Características:

  • Segue padrões W3C (Contendo diferença de interpretação)
  • Gratuitos

Aplicação Web

Definição: Soluçao/produto que utiliza a internet como meio de comunicação entre Cliente x Server e sem precisar realizar a instalação da aplicação.
EX: YouTube, Linkedin ou Facebook (desde que usados pelo navegador).

Definição de Server:

Lado em que se fornece serviços/conteúdo da internet ou em uma LAN(Local Area Network). Ao restringir acesso a rede externa a um servidor ele atua em uma INTRANET.

Geralmente são máquinas mais robustas, o conjunto de servidores é denominado de Data Center.

Principais serviços oferecidos são:

  1. Arquivos
  2. Segurança (proxy/firewall)
  3. Streaming
  4. E-mail
  5. Web

Proxy

Definição: serviço que fornece regras de acesso a internet. Função de restringir a navegação a determinados sites em uma rede local. Regras são estabelecidas pela empresa/organização que implementou.

Firewall

Definição: Servidor dedicado a segurança que faz o controle de acesso e informações recebidos externamente em uma rede local.

Servidor Web

Definição: Responsável por tratar requisições HTTP ( feitas pelo cliente ) e encaminhar respostas/arquivos correspondentes.

Como é uma estrutura básica de um HTML?

Definição:

HTML é uma linguagem de marcação utilizada para estruturar e organizar semanticamente elementos em um página web.

O principal elemento de marcação e a Tag, sendo sempre demilitado pelos simbolos <> para abertura e para fechamento, sendo esse último opcionais em alguns elementos.

Geralmente o arquivo HTML principal de um página é nomeado como index.html

Estrutura básica de um arquivo HTML

  1. Tag html
    • Engloba todos os elementos
  2. Tag head
    • carregamento de metadados ( não visiveis ao usuário geralmente).
      EX:Titulo da pág.links de estilo e scripts, idioma.
  3. tag body
    • Elemento visiveis do site.
      EX:Texto, imagens, links

OBS: Uma tag pode possuir atributos, podendo ser mais especificos ou mais genéricos.
EX: atributo id e atributo type da tag input.
